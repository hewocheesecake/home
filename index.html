Minesweeper.

Minesweeper is a computer game that challenges players to navigate a minefield and uncover hidden bombs by placing a flag on them. The game has set level size and difficulty levels increase with the number of mines that appear in each level. Players of all skill levels can enjoy this game. Minesweeper is an excellent way to pass the time and challenge your brain. It is only available on computers.
